AFE_Examples.eww is an IAR workspace file which contains all of the
Analog Front End (AFE) related example projects for the ADuCM350 in
a single workspace file.

The workspace contains the following projects:
 - AfeTest
 - AfeCustomInterrupt
 - AmperometricMeasurement
 - AuxChanMeasurement
 - ImpedanceMeasurement_2Wire
 - ImpedanceMeasurement_4WireBioIsolated
 - UnipolarCurrentMeasurement

To build any of the projects, it is first necessary to make this project
the active project.  Do this by right clicking the project name in the
"Workspace" window and select "Set as Active".

Note that all of the above projects also exist as standalone workspace
examples, located in their respective directories.
