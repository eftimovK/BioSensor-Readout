ADI_INT_STATUS adi_int_EnableInt ( uint32_t  iid,
bool  bEnable 
)

Enable or disable the interrupt.

This API enables or disables the spacified interrupt. If an Interrupt is disabled then the interrupt handler will not be run even if the interrupt is raised

Parameters
[in]iidInterrupt ID.
[in]bEnableFlag which indicates whether to enable or disable the given Interrupt.

This argument should be set to:
  • 'true' to enable the interrupt.
  • 'false' to disable the interrupt.
Returns
  • ADI_INT_SUCCESS if successfully enabled or disabled the interrupt.
  • ADI_INT_FAILURE if the interrupt could not be enabled or disabled.

ADI_INT_STATUS adi_int_InstallHandler ( uint32_t  iid,
ADI_INT_HANDLER_PTR  pfHandler,
void *  pCBParam,
bool  bEnable 
)

Install the handler for a Coretex-M3 Interrupt.

This API can be used to install an interrupt handler for a given system interrupt.

Parameters
[in]iidInterrupt ID. For system interrupts this is the system interrupt ID. This will be a unique number to identify a particular interrupt. Please refer to the Hardware Reference Manual to find the number for a specific interrupt.
[in]pfHandlerHandler which handles the interrupt. This handler will be called when the interrupt occurs.
[in]pCBParamApplication provided callback parameter which will be passed back when the given handler is called.
[in]bEnableFlag which indicates whether to enable/disable the given Interrupt after the interrupt handler installation. The interrupt can be disabled during the installation and enabled later by calling the function adi_int_EnableInt().

This argument should be set to:
  • 'true' to enable the interrupt.
  • 'false' to disable the interrupt.
Returns
  • ADI_INT_SUCCESS if successfully installed the interrupt handler.
  • ADI_INT_FAILURE if it was unable to install the interrupt handler.
