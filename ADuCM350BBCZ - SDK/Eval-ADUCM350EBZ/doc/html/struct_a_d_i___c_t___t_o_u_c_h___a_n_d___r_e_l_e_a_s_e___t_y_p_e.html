Detailed Description

CT Touch And Release configuration Struct. The Touch and Release data type struct is used with the adi_CT_SetTouchCfg(), adi_CT_GetTouchCfg(), adi_CT_SetReleaseCfg() and adi_CT_GetReleaseCfg() APIs to set or get the Touch and Release Detection Algorithm settings.
