Detailed Description

CT Timing Settings Structure. The Timing data type struct is used with the adi_CT_SetTimingsCfg() and adi_CT_GetTimingsCfg() APIs to set or get the timing conversion settings. AutozeroDuty ADI_CT_AUTOZERODUTY_TYPE C2VHoldTime Capaticitance to Voltage Time. 5 bits, each LSB represents 0.25us, range [0.25-8]us. Phase13 Phase13. 5 bits, each LSB represents 0.25us, range [0.25-8]us. Phase24 Phase24. 8 bits, each LSB represents 0.25us, range [0.0625-16]us.
